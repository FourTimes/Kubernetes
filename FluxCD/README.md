# FluxCD Install
   
      wget -q https://github.com/fluxcd/flux/releases/download/1.20.2/fluxctl_linux_amd64
      mv fluxctl_linux_amd64 fluxctl
      chmod +x fluxctl
      sudo mv fluxctl /usr/local/bin/
      fluxctl version
      
 **Requirements**
 
    1. kubernetes cluster
    2. fluxctl
    3. Repositary
    4. Docker image ( Public / Private Repositary)
